# Web CV Profesional - Muhammad Afdal Farid

Website CV Dark Mode dengan fitur auto-generate PDF.

## Fitur
- **Responsive Navigation**: Menu hamburger untuk mobile.
- **Client-Side PDF Export**: Mengubah tampilan web menjadi PDF secara instan.
- **Dark Mode**: Desain elegan dengan aksen biru laut.

## Instalasi
1. Letakkan semua file dalam satu folder.
2. Siapkan `profil.jpg`, `favicon.ico`, dan `thumbnail.jpg`.
3. Buka `index.html` di browser.
